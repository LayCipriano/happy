## Como Executar o Projeto
Clone este repositório:
~~~bash
git clone https://github.com/seu-usuario/seu-repositorio.git
~~~
Acesse a pasta do projeto:
~~~bash
cd seu-repositorio
~~~
Instale as dependências:
~~~bash
npm install
~~~
Inicie o projeto:
~~~bash
npm start
~~~
O site estará disponível em http://localhost:3000.
